# Setup
Clone the repository, then open a terminal in the repository folder (iteration-2-build-with-style-group-2). Run `npm i` to install all of the required dependencies (such as React and Vite build tools)

# Testing for development
Run `npm run dev` in the terminal to start a development server. Once the development server is started, go to `http://localhost:3000` in your browser. Any changes you make will automatically update in your browser, so don't bother reloading the page unless there's an issue.

# Deploying
Run `npm run build` to generate static HTML files. These files will be stored in a folder called "dist".
